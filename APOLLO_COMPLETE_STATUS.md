## üöÄ Deployment Instructions

### Prerequisites
- Docker & Docker Compose
- Node.js 20+ (for local development)
- Python 3.11+ (for local development)
- Git

### Quick Start (Production)

```bash
# 1. Clone repository
git clone <apollo-repo>
cd apollo

# 2. Configure environment
cp .env.example .env
# Edit .env with your settings (database passwords, API keys, etc.)

# 3. Start infrastructure (databases)
cd infrastructure/databases
docker-compose up -d

# Wait for databases to initialize (2 minutes)
docker-compose ps  # Check all services are healthy

# 4. Start backend services
cd ../../services
docker-compose up -d

# 5. Start frontend
cd ../frontend/web-console
docker-compose up -d

# 6. Start intelligence services
cd ../../intelligence
docker-compose up -d

# 7. Start blockchain forensics
cd ../intelligence/blockchain-forensics
docker-compose up -d

# 8. Verify all services
curl http://localhost:3000/health  # API Gateway
curl http://localhost:3001/health  # Auth Service
curl http://localhost:8000/health  # Intelligence API
curl http://localhost:8001/health  # Blockchain API

# 9. Access web console
open http://localhost  # or http://localhost:80
```
